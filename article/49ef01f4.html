AI编程体验 GitHub Copilot

发表于 2022-04-08 分类于 编程

GitHub Copilot

前言：

GitHub Copilot是微软与OpenAI共同推出了一款AI编程工具，基于 GitHub 及其他网站的源代码，可根据上文提示为程序员自动编写下文代码。

这可谓是程序员的福音，根据项目官方页面显示，Copilot 可以支持十几种语言，与 Python、JavaScript、TypeScript、Ruby 和 Go 等主流语言配合效果更佳。

因此为了提升Code的幸福感，我按照提示注册申请，一天左右电子邮箱里就提示可以开始使用了。那下面开始体验吧！

编程的好助手：

最近几年，互联网大厂的面试中少不了的就是算法。最近我在刷LeetCode的过程中发现，自己也常常会因为自身水平的因素限制不能熟练掌握每一种算法。这里拿一道简单的LeetCode题目，让Copilot试试：

两数之和：

给定一个整数数组 nums 和一个整数目标值 target，请你在该数组中找出 和为目标值 target 的那 两个 整数，并返回它们的数组下标。

你可以假设每种输入只会对应一个答案。但是，数组中同一个元素在答案里不能重复出现。

你可以按任意顺序返回答案。

示例 ：

输入：nums = [2,7,11,15], target = 9
输出：[0,1]
解释：因为 nums[0] + nums[1] == 9 ，返回 [0, 1] 。

编程的未来？

也许有人会说，如果有一天它足够智能，程序员不就会面临失业的风险吗？

我想说也许不会。因为Copilot可以帮助到你的是重复性的工作，而程序员更需要专注于的是创造性的工作。

用另一句话来讲，Copilot帮助了你减少从Google和GitHub上搜索源代码、解决方法的时间，至于到真正需要创新的地方，一定还是需要各位程序员的创造思维。
